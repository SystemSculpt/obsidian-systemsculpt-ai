# Styling token reference

Last verified against `src/css/base/variables.css`: **2026-02-11**.

## Core semantic tokens

- Primary: `--ss-color-primary`, `--ss-color-primary-hover`
- Secondary/surface: `--ss-color-secondary`, `--ss-color-bg-primary`, `--ss-color-bg-secondary`
- Text: `--ss-color-text-normal`, `--ss-color-text-muted`, `--ss-color-text-accent`
- Borders: `--ss-color-border`, `--ss-color-border-hover`
- State colors: `--ss-color-success`, `--ss-color-warning`, `--ss-color-danger`

## Layout and typography tokens

- Spacing: `--ss-space-xxs` through `--ss-space-xxl`
- Font sizes: `--ss-font-size-xs` through `--ss-font-size-xl`
- Font weights: `--ss-font-weight-normal`, `--ss-font-weight-medium`, `--ss-font-weight-bold`
- Radius: `--ss-radius-sm`, `--ss-radius-md`, `--ss-radius-lg`

## Motion and depth tokens

- Transitions: `--ss-transition-fast`, `--ss-transition-normal`, `--ss-transition-slow`
- Shadows: `--ss-shadow-sm`, `--ss-shadow-md`, `--ss-shadow-lg`
- Surface/overlay helpers: `--ss-layer-*`, `--ss-border-*`, `--ss-shadow-*`

## Notes

- Tokens are mapped from Obsidian variables where possible.
- Token values are intended to adapt to the active Obsidian theme.
