# SystemSculpt 6.0.2

SystemSculpt 6.0.2 hardens license validation and makes future releases visible inside Obsidian.

- A temporary or malformed validation response can no longer be mistaken for an invalid paid license.
- Only an exact server rejection can clear a previously valid license. Connection failures preserve the last known state and show a retryable availability message.
- First-time activation retains the entered key during a temporary outage, so Retry remains available.
- New releases are announced in Obsidian, remain visible in the status bar until installed, and open the official Community Plugins page for installation.
- A manual `Check for updates` command and post-update confirmation make the installed state explicit.
