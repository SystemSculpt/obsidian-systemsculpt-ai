# SystemSculpt 6.0.0

SystemSculpt 6 replaces the old client-owned provider stack with one managed desktop architecture.

- Chat, transcription, embeddings, and Studio use SystemSculpt services with no provider-key setup.
- Provider catalogs, model selection, local Pi execution, custom endpoints, and custom HTTP MCP configuration were removed.
- Existing managed chats continue normally. Chats tied to retired Pi or external-provider metadata open read-only.
- Settings migration removes retired credentials while preserving the SystemSculpt license, vault folders, Chat favorites, Studio projects, recorder preferences, embeddings, and workflows.
- The plugin is now desktop-only.
