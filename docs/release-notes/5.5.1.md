# SystemSculpt 5.5.1

## What's New

SystemSculpt 5.5.1 is a small Studio polish and update-safety patch. Studio node selection now reads more clearly when a node has expanded controls, and the release path has stricter checks so future updates are less likely to ship with mismatched version files or missing plugin assets.

## Highlights

- Studio now wraps the selected and hovered node glow around expanded top and bottom controls, not just the center card.
- Studio no longer uses the old left-edge run-status stripe, so selection and hover states are easier to read.
- Release checks now verify the plugin version surfaces, public release notes, ignored build artifacts, and the exact Obsidian asset set before publishing.

## Upgrade Notes

- No manual action is needed.
- Existing chats, settings, and Studio projects should continue to load normally.
- Minimum supported Obsidian version remains `1.4.0`.

## Fixes and Reliability

- Fixed selected Studio nodes with expanded controls so the accent border reads as one continuous shape.
- Improved hover and selection feedback while keeping active node dragging lighter.
- Added release-surface validation so future patch releases catch version drift and missing assets before a GitHub release is created.
