# SystemSculpt 5.9.0

## What's New

SystemSculpt 5.9.0 is a large feature and reliability release. It expands
bring-your-own-key support to Claude and Gemini through their native APIs, makes
BYOK users first-class (no license wall in chat), hardens startup and updates on
every platform, and overhauls the embeddings and audio-transcription pipelines.
It also lands the clearer license-renewal experience and a batch of chat and
agent-mode improvements.

## Highlights

### Providers (bring your own key)

- **Claude (Anthropic) BYOK** via the native Messages API — add your Anthropic
  key and use Claude models directly in Chat.
- **Gemini (Google) BYOK** via the native `generateContent` API.
- A connected provider's models can no longer silently disappear from the Chat
  picker: a model-catalog status state machine keeps the dropdown honest.
- **BYOK users never hit a license wall.** When you run on your own provider key,
  chat no longer demands a SystemSculpt license, and a provider-key auth failure
  is reported as a provider problem ("Open Providers") rather than a
  subscription/renewal prompt.

### Platform reliability

- More reliable startup on Android and Linux, including a Node-free mobile
  bundle-load path so the plugin initializes without desktop-only APIs.
- A disabled plugin is now fully inert — no timers, listeners, or background work
  linger after you turn it off.
- Versioned settings migration with update safety: settings are migrated
  forward deterministically, with safe-mode and minimum-version fail-soft so a
  bad load can't wedge the plugin.

### Embeddings

- A portable embeddings index that survives Obsidian Sync and vault
  restore — export/import, snapshots, and automatic restore.
- Typed provider errors and more robust response parsing.
- Automatic retry with backoff for transient provider errors and rate limits.
- A bulk rebuild that can resume after an interruption even when auto-processing
  is off.

### Recording & transcription

- A typed transcription provider core: typed errors, a response normalizer, and
  an HTTP classifier shared across providers.
- **Self-hosted Whisper** support with config-time validation and a settings UI.
- A separate, configurable post-processing model for cleaning up transcripts.
- Captured audio is saved when iOS interrupts a recording (lock screen / phone
  call), so meetings aren't lost.
- Mobile recording bitrate is capped so long meetings stay within transcription
  limits.
- One-tap stop for the recorder — the stop control no longer needs repeat taps.
- YouTube transcription now returns an actionable message when the transcript
  service is over quota, instead of leaking a raw error.

### Chat & agent mode

- Agent mode is scoped to the individual chat instead of a global setting.
- The local Pi agent loop stops cleanly on repeated identical failing tool calls
  and surfaces an actionable error when a tool-use turn stalls.
- Favorite models are restored in the composer model picker.
- Filesystem agent tools run on mobile, and Folder Notes paths resolve correctly
  across all file tools.
- A per-chat toggle to hide system and tool messages.
- "Attach Context File" is available from the slash palette.

### License experience

- A clear, actionable renewal flow for expired or invalid SystemSculpt
  licenses, replacing the old generic stream error — with distinct copy and
  actions for expired vs. invalid vs. provider-key states.

## Upgrade Notes

- No manual action is required; settings migrate automatically on update.
- Minimum supported Obsidian version remains `1.4.0`.
- BYOK users: if you previously saw a license prompt while using your own
  provider key, that no longer happens — chat runs on your key.
